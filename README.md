# Study Planner

A basic study planner built with HTML, CSS, and JavaScript.

## Features

* Add subjects
* Add assignments
* Track assignment completion
* Progress bars
* Local Storage support

## How to Run

Either:
1. Download or clone the repository.Open project folder and Open `index.html` in a web browser.
   
   or
   
3. Open the project in VS Code and run it using Live Server.

## Technologies Used

* HTML
* CSS
* JavaScript
